# <center>CommPass</center>

#### <center> *Finding Common Ground Within Our Communities*</center>  
*<center> A DiamondHacks 2026 Project </center>*
<br>
A justice-focused platform that helps people in high-burden (CalEnviroScreen “red”) areas find nearby support systems.  
<br>
Common Ground is a platform that connects high-burden communities to nearby resources like libraries, language programs, and UC Links initiatives, to turn existing data into real opportunities. Communities facing the highest environmental burdens often have the least access to educational and technological resources.

## Features
- Identifies environmentally burdened communities using CalEnviroScreen
- Connects users to local educational and community resources
- Highlights programs like UC Links

## How it works
1. Enter your ZIP code in the search bar and click "Explore".
2. You will get a result of how high impact your area is.
3. You can start exploring the resources available in vicinity of your area!
